# タスク管理アプリ
チームでのタスク管理ができるアプリ

## 特徴
各ユーザーは以下の操作が可能

**管理者:**
-   タスクの作成、ユーザーへのタスクの割り当て
-   ユーザー一覧の閲覧、ユーザーの削除
-   タスク一覧の閲覧、タスクの編集・削除
-   タスクの完了・未完了の切り替え
-   プロフィールページの閲覧

**一般ユーザー:**
-   タスクの作成
-   ユーザー一覧の閲覧
-   タスク一覧の閲覧、担当しているタスクの編集・削除
-   担当しているタスクの完了・未完了の切り替え
-   プロフィールページの閲覧

## 開発環境
* Java 8（jdk1.8.0_171）
* Spring Boot 2.4.3
* H2 database
* Maven 3.6.3
* Bootstrap 4.2.1
* jQuery 3.3.1-1
